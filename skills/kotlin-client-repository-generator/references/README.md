# Kotlin Client Repository Generator References

This directory stores extended materials for `kotlin-client-repository-generator`.

- `examples.md`: Full compile-ready templates and scenario snippets (DAO-only, HTTP-only, DAO+HTTP, Flow, HTTP 204/no-body, validation bad cases).

Usage guidance:
- Keep `SKILL.md` concise and canonical.
- Keep long code templates and expanded examples in `references/examples.md`.
- When updating rules in `SKILL.md`, update related examples in `references/examples.md` in the same change.
